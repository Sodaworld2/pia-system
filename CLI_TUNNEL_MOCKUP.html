Technical Logic:

The PIA Tunnel script (pia-tunnel.mjs) spawns Claude as a child process. It uses STDIO Pipes to read every character Claude prints and broadcast it via WebSockets. When you type in the remote input, PIA receives the message and writes it directly to Claude's stdin, fooling Claude into thinking there's a physical keyboard connected at home.
