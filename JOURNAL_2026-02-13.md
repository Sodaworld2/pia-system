**Screenshot utility**: `take-ss.cjs` deployed on Machine #3

- Saves to PIA's `public/screenshots/` directory (served via HTTP)
- Usage: `node take-ss.cjs <name>` → saves as `screenshots/<name>-<timestamp>.png`
- Retrievable from Machine #1 via: `curl http://100.102.217.69:3000/screenshots/<filename>.png`

**File transfer pattern** (Machine #1 → Machine #3 via PTY):
1. Write script locally
2. Base64 encode: `base64 -w 0 script.cjs`
3. Send via PTY: `node -e "require('fs').writeFileSync('path', Buffer.from('B64','base64').toString())"`
4. Execute: `node path/to/script.cjs`
